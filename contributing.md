# Contributing

## Cards

Cards can be added and modified using this file `src/housing_laws/config.yaml`

## One Pagers

The 'One-Pagers' for each law can be modified in it's perspective folder under 'src/housing_laws/[law]'.  There are two files:

1. **qa.yaml** - The question and answer section in yaml
2. **readme.md** - The one pager, written in [markdown](https://github.com/adam-p/markdown-here/wiki/markdown-cheatsheet)
